# ایندکس اصلی ممیزی

- Roadmap
  - `roadmap/roadmap_master.md`

- Dashboard (offline)
  - `dashboard/index.html`
  - `dashboard/data/roadmap_data.json`
- Logs
  - `logs/actions.log`
  - `logs/errors.log`

## نکات اجرایی
- داشبورد به‌صورت آفلاین با باز کردن مستقیم `dashboard/index.html` قابل مشاهده است.
- inventory سازمان GitHub در این اجرا به‌دلیل نبود `gh` ناقص است و در لاگ خطا ثبت شده.

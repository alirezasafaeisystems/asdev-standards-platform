# نمای کلی پروژه asdev-standards-platform

## هدف
این مخزن به‌عنوان هاب استانداردسازی، اتوماسیون حاکمیت و ارکستراسیون گزارش‌گیری برای ریپازیتوری‌های ASDEV عمل می‌کند.

## وضعیت فعلی
- بلوغ اتوماسیون: **بالا** (Make targets متعدد برای lint/test/verify/reports).
- ریسک عملیاتی: **متوسط** (وابستگی به ابزار بیرونی مانند `gh` و `yq`).
- پوشش CI: **متوسط رو به بالا** (workflow زمان‌بندی‌شده + PR/Push).

## شواهد
- دستورات اصلی عملیات در README تعریف شده‌اند (`make lint/test/verify/reports`).
- داشبورد مدیریتی محلی با URL پیش‌فرض مشخص دارد.

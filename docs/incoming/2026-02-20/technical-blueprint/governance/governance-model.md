# Governance & Versioning Model

## Versioning

SemVer (MAJOR.MINOR.PATCH)

## Release Policy

- Major: breaking policy changes
- Minor: new standards/features
- Patch: fixes

## Required Files

- CHANGELOG.md
- SECURITY.md
- CONTRIBUTING.md
- SUPPORT.md

## Governance Principles

- Transparent changes
- Mandatory review
- Immutable tagged releases

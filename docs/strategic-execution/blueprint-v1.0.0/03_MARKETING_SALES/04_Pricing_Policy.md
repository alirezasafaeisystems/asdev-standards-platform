# سیاست قیمت‌گذاری (اجرایی و قابل دفاع)

## اصول
- قیمت بر اساس «ریسک کاهش داده‌شده» و «هزینه شکست» تعیین می‌شود، نه صرفاً ساعت کار.
- هر Offer دارای Scope ثابت است.
- هر مورد خارج از Scope → Change Request.

## ساختار پرداخت پیشنهادی
- 40% شروع
- 40% بعد از Delivery Gate
- 20% بعد از Post-Delivery Review

## تخفیف/توافق
- فقط برای قراردادهای بلندمدت یا Referral معتبر
- هر تخفیف باید دلیل داشته باشد و ثبت شود.

# فرآیند فروش و قرارداد (Stage-Gated)

## Gate S0 — Qualification (ورود)
- ورودی: فرم یا معرفی
- خروجی: Qualified / Not Qualified
- Evidence: فرم ثبت‌شده + یادداشت اولیه

## Gate S1 — Discovery Call
- هدف: فهم مشکل واقعی، محدودیت‌ها، و ROI
- خروجی: Discovery Notes + Next Step
- Evidence: `05_TEMPLATES/06_Discovery_Notes.md`

## Gate S2 — Technical Assessment (سبک)
- هدف: ارزیابی حداقلی ریسک‌ها و scope
- خروجی: Assessment Summary
- Evidence: `05_TEMPLATES/07_Assessment_Summary.md`

## Gate S3 — Proposal + SOW
- خروجی: Proposal + Scope + قیمت + تعهدات
- Evidence: فایل امضا/تایید

## Gate S4 — Contract
- خروجی: قرارداد + پرداخت مرحله اول
- Evidence: قرارداد + رسید/تایید

## Gate S5 — Delivery
- خروجی: Handover Pack + Final Report
- Evidence: `05_TEMPLATES/04_Final_Report.md` + پیوست‌ها

## Gate S6 — Post-Delivery Review
- هدف: کاهش ریسک پس از تحویل، دریافت testimonial
- Evidence: `04_OPERATIONS/03_Post_Delivery_Review.md`

# جزئیات سه پکیج (قابل ارائه روی سایت)

## Offer 1 — Quick Stability Audit
### برای چه کسانی؟
کسب‌وکارهایی که سایت/سیستم دارند ولی کندی، قطعی، خطای پرداخت، یا نااطمینانی دارند.
### تحویل می‌گیرید
- گزارش رسمی ریسک‌ها و اولویت اصلاح
- 3 اقدام اصلاحی کلیدی (با scope مشخص)
- چک‌لیست پایداری و امنیت پایه
### خروجی‌ها
- Risk Report + Action Plan + Evidence

---

## Offer 2 — Local Resilience Program (Core)
### برای چه کسانی؟
سازمان‌هایی که نیاز به استقرار داخلی، کاهش وابستگی خارجی، و پایداری در شرایط اینترنت ناپایدار دارند.
### تحویل می‌گیرید
- طراحی معماری داخلی/تاب‌آوری
- حذف وابستگی‌های حیاتی خارجی (یا جایگزین داخلی)
- استقرار + مانیتورینگ پایه + لاگ/گزارش
- مستند رسمی قابل دفاع
### خروجی‌ها
- Architecture Note + Deployment Evidence + Runbook + Final Report

---

## Offer 3 — Project Rescue to Production
### برای چه کسانی؟
پروژه‌های نیمه‌کاره یا لانچ‌شده‌ی بی‌ثبات که نیاز به مهندسی دوباره دارند.
### تحویل می‌گیرید
- بازمهندسی مسیر release
- استانداردسازی و quality gates
- اصلاح معماری/عملیات برای production
### خروجی‌ها
- Production Readiness Report + Go/No-Go Checklist + Handover

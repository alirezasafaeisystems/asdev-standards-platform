# KPI و متریک‌ها (حداقلی و اجرایی)

> هدف: سنجش بدون پیچیدگی. هر KPI باید قابل اندازه‌گیری باشد.

## Brand & Proof
- تعداد Case Study های منتشر شده (هدف: 3–5)
- تعداد مقاله/پست‌های تخصصی (هدف: حداقل 6 محتوای مرجع)

## Portfolio Funnel
- CTR CTA (Home → Service Page)
- نرخ شروع فرم (Service Page → Form Start)
- نرخ تکمیل فرم (Form Start → Submit)
- نرخ Qualified Lead (Submit → Qualified)

## Sales
- تعداد Discovery Call
- نرخ تبدیل Discovery → Proposal
- نرخ تبدیل Proposal → Contract
- متوسط ارزش قرارداد
- زمان چرخه فروش (Lead → Contract) *بدون تعهد زمانی، فقط ثبت*

## Delivery / Operations
- درصد تحویل‌های دارای Evidence کامل
- تعداد Incidents بعد از تحویل (۳۰ روز اول)
- تعداد Change Requests خارج از Scope

## SEO (حداقل)
- وضعیت Indexation (Search Console)
- تعداد صفحات indexed (Portfolio + PersianToolbox)
- impressions/clicks (trend)

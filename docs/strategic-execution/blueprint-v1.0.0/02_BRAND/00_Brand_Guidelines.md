# Brand Guidelines (نسخه اجرایی)

## لحن (Tone)
- متعادل، حرفه‌ای، قابل فهم
- بدون اصطلاحات سنگین
- بدون اغراق و شعار

## پیام‌سازی
- مشکل را ساده بگو: «قطعی، تحریم، وابستگی خارجی، پایداری»
- نتیجه را واضح بگو: «استقرار داخلی، گزارش‌پذیری، کاهش ریسک»

## عناصر اعتماد (Trust)
- نمونه‌کار واقعی
- گزارش رسمی
- فرآیند استاندارد تحویل
- شفافیت در Scope و هزینه

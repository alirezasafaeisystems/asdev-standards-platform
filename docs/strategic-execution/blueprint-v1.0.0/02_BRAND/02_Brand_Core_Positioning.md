# Brand Core — Positioning (Draft v1.0)

## One-liner
«من سیستم‌های وب را برای شرایط واقعی ایران (تحریم/فیلتر/اینترنت ناپایدار) پایدار و داخلی‌سازی می‌کنم.»

## برای چه کسی؟
کسب‌وکارهایی که درآمد یا عملیات‌شان به سرویس آنلاین وابسته است و نمی‌خواهند با قطع سرویس خارجی یا قطعی اینترنت آسیب جدی ببینند.

## چه چیزی تحویل می‌دهم؟
- استقرار داخلی و بدون وابستگی حیاتی
- گزارش‌پذیری و شفافیت عملیات
- مسیر اصلاح پروژه‌های نیمه‌کاره تا لانچ قابل دفاع

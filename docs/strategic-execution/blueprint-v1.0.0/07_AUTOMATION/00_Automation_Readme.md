# Automation & Logs

این پوشه برای «ثبت و گزارش‌گیری» است؛ حتی اگر همه چیز دستی انجام شود.

## فایل‌های log
- `Outreach_Log.csv`: ارتباط‌گیری‌ها و نتایج
- `Content_Log.csv`: محتواهای منتشرشده
- `Lead_Log.csv`: لیدها و وضعیت
- `Project_Log.csv`: پروژه‌ها و گیت‌ها
- `Weekly_Reviews/`: گزارش‌های هفتگی
- `GoLive_Evidence/`: شواهد گولایو

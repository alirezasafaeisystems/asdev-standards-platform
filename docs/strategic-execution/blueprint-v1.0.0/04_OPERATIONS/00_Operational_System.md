# سیستم عملیات (Operating System)

## هدف
اجرای قابل کنترل، گزارش‌پذیر، و قابل ممیزی برای پروژه‌ها و فعالیت‌های برند/فروش.

## اجزا
- Task Board (stage-gated)
- Evidence Policy (بدون evidence هیچ Done نداریم)
- Weekly Review (گزارش ثابت)
- Project Handover Pack (تحویل استاندارد)
- Change Request Process

# سیاست Evidence (Definition of Done)

## چرا؟
بازار ایران ممکن است کیفیت را ابتدا نفهمد، اما «مدرک» را می‌فهمد.
Evidence هم از دوباره‌کاری جلوگیری می‌کند هم برند می‌سازد.

## برای هر تسک باید وجود داشته باشد
- Owner
- Output artifact (فایل/لینک)
- Acceptance Criteria (قابل تست)
- Evidence Link/Attachment

## انواع Evidence
- لینک production / staging
- اسکرین‌شات
- گزارش PDF/MD
- لاگ تست/مانیتورینگ
- تایید مشتری (ایمیل/پیام)

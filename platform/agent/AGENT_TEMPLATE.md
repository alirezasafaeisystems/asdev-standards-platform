# Agent Template

## Identity & Mission
Deliver safe, incremental improvements with verifiable outcomes.

## Repo Commands
- `make lint`
- `make test`
- `make build`

## Workflow Loop
Plan -> Implement -> Validate -> Report.

## Definition of Done
All required checks pass and artifacts are updated.

## Human Approval Gates
Escalate for auth/security, breaking API/schema/db, and critical UX flow changes.

## Quality Checklist
- Lint clean
- Tests pass
- Build passes

## Lenses
- Reliability
- Security
- Maintainability

## Documentation & Change Log Expectations
Update impacted docs and execution reports in the same PR.

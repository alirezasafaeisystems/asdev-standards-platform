# Repo Lens Selection Rules

Enable only lenses relevant to the repository domain.

## Baseline Lenses (all repos)

- Quality
- Reliability
- Security
- Documentation

## Web/Product Repos

Enable additionally:

- UX/Accessibility
- SEO/Performance
- Product behavior

## Payment/Financial/Monetization Repos

Enable additionally:

- Legal/Compliance
- Risk/Auditability
- Abuse/Fraud resistance

## Language Pilot Repos (Go/Python)

Enable additionally:

- Simplicity/Maintainability
- CI parity with Level 1 standards

## Governance Hub Repos

Enable additionally:

- Template traceability
- Cross-repo adoption safety
- Policy coherence
